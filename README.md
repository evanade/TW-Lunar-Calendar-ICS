此文件是參考[Javascript農民曆萬年曆](http://dolphin.cc.ncu.edu.tw/lunar.html)的程式碼  
把程式碼作一些修改和延伸  
印出簡略版的iCalendar格式的文件  
然後再同步到Google變成Googel標準格式  

此農曆行事曆包含：  
農曆日期  
二十四節氣  

檔案部分格式如下

    SUMMARY:七月初九 
    DESCRIPTION:
     
    SUMMARY:白露 
    DESCRIPTION:

以下提供農曆.ics檔案分享連結

可以直接訂閱或下載使用

Android、iPhone等智慧型手機也可以訂閱使用
